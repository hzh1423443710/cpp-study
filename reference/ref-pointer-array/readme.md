### 引用和指针的区别

- 引用必须初始化 指针可以不用
- 引用的赋值是赋值 指针的赋值是修改指向

- 存在指针的引用,数组的引用,函数的引用



### 引用不是对象,不占空间,没有地址

- **没有指向引用的指针**

```c++
int v = 1;
int &ri = v;
int& *p = &ri;//× 没有指向引用的指针 没有这种写法
```

- **没有引用的引用**

```c++
int v = 10;
int& r = v;
int& &rr = r;	//×
int& rr = r;	//普通引用
```

- **没有引用的数组**

```c++
int a = 1, b = 2, c = 3;
int& arr[3] = {a,b,c};//×
```
